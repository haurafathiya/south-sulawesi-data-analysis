# 📊 South Sulawesi Socioeconomic Data Analysis

## Project Overview

This project was completed as part of the *Introduction to Data Science* course. It explores socioeconomic indicators across regencies and cities in South Sulawesi using statistical analysis and machine learning techniques.

## Dataset

The dataset contains 57 observations representing regencies and cities in South Sulawesi with the following variables:

- Expected Years of Schooling (HLS)
- Human Development Index (HDI/IPM)
- Gross Regional Domestic Product (GRDP/PDRB)
- Other socioeconomic indicators
- Classification label

## Methods

- Data preprocessing
- Pearson Correlation Analysis
- Euclidean Distance Calculation
- K-Nearest Neighbors (KNN)
- Data Visualization
